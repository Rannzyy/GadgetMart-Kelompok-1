# GadgetMart.id

GadgetMart.id adalah platform e-commerce berbasis web yang dibuat buat bantu user cari, pilih, dan beli gadget dengan lebih gampang. User bisa lihat detail produk mulai dari spesifikasi, harga, sampai stok tanpa ribet.

Tampilannya simpel, navigasi jelas, ada fitur keranjang dan checkout biar proses beli jadi lebih praktis tanpa harus ke toko langsung.

Selain itu, sistem juga support role admin buat kelola produk dan transaksi.

---

## Fitur Utama

* Login & Logout user
* Role admin & user
* CRUD produk (tambah, edit, hapus)
* Manajemen stok barang
* Keranjang belanja
* Checkout / transaksi
* Riwayat transaksi
* Cek stok otomatis
* Notifikasi stok tidak cukup

---

## Teknologi yang Dipakai

* PHP Native
* MySQL
* HTML, CSS, JavaScript
* XAMPP

---

## Cara Menjalankan Project

1. Pindahkan folder project ke:

```
htdocs/GadgetMart.id
```

2. Jalankan XAMPP (Apache & MySQL)

3. Import database:

* buka phpMyAdmin
* buat database: `gadgetmart`
* import file: `gadgetmart.sql`

4. Akses di browser:

```
http://localhost/GadgetMart.id/user/login.php
```
